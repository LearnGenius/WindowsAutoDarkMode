# Usage
## A. 手动执行
到Release界面下载bat文件，双击运行
## B. 加入任务计划程序自启动
1. 到Release界面下载bat文件，随便存放在一个位置
（尽量非系统盘，避免出现运行权限问题）
2. 按下 <kbd>Win</kbd> + <kbd>R</kbd> 打开运行对话框。
3. 键入 `taskschd.msc` 然后按 <kbd>Enter</kbd>。
4. 创建任务
   1. 常规：
      1. 名称随便填
      2. 描述随便填
      3. 安全选项仅勾选：不管用户是否登录都要运行
   2. 触发器：
      1. 开始任务：按预定计划：按需设置
      2. 开始任务：登陆时：按需设置
   3. 操作：
      1. 操作：启动程序
      2. **程序或脚本：添加本项目文件**
   4. 条件：全部取消勾选
   5. 设置：按需设置
5. 完成。
